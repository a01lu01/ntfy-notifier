# ntfy Windows 托盘通知工具

## 项目概述

| 项目 | 内容 |
|------|------|
| 项目名称 | ntfy-Notifier |
| 核心功能 | 监听 ntfy 消息并弹出 Windows 系统通知 |
| 目标用户 | 需要在 PC 上接收验证码通知的用户 |

---

## 功能需求

### 核心功能

| 功能 | 优先级 | 说明 |
|------|:------:|------|
| 系统托盘图标 | P0 | 常驻后台运行，最小化到托盘 |
| 托盘菜单 | P0 | 设置、退出、关于 |
| 设置窗口 | P0 | 配置服务器、账号、话题 |
| Windows 通知 | P0 | 收到消息弹通知 |
| 开机自启 | P1 | 可选开机启动 |

### 设置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 服务器地址 | http://114.55.43.156:8080 | ntfy 服务端地址 |
| 用户名 | 空 | 登录用户名 |
| 密码 | 空 | 登录密码 |
| 订阅话题 | 空 | 要订阅的话题 |
| 轮询间隔 | 3 秒 | 检查新消息的间隔 |
| 开机自启 | 否 | 是否开机自动启动 |

## 交互流程

```
┌──────────────┐
│   启动程序    │
└──────┬───────┘
       ↓
┌──────────────┐
│ 显示托盘图标 │
└──────┬───────┘
       ↓
┌──────────────┐
│ 读取配置文件  │ ← 首次：无配置文件 → 弹出设置窗口
└──────┬───────┘
       ↓
┌──────────────┐
│ 连接 ntfy   │ → 连接成功 → 开始轮询
└──────┬───────┘
       ↓
┌──────────────┐
│  收到消息    │
└──────┬───────┘
       ↓
┌──────────────┐
│ 弹出通知    │
└──────────────┘
```

## 配置存储

使用 JSON 文件存储配置：

```json
{
  "server": "http://114.55.43.156:8080",
  "username": "iPhone",
  "password": "WHYntfy2026",
  "topic": "sms",
  "poll_interval": 3,
  "auto_start": false
}
```

配置文件位置：`%APPDATA%/ntfy-notifier/config.json`

---

## 打包发布

使用 PyInstaller 打包：

```bash
pyinstaller --onefile --icon=icon.ico ntfy_notifier.py
```

输出单个 exe 文件，用户双击即可运行。

---

## 风险与注意事项

| 风险 | 应对 |
|------|------|
| 通知权限被拒 | 提示用户开启通知权限 |
| 服务器连接失败 | 显示托盘状态为"未连接" |
| 密码明文存储 | 提示用户注意安全性 |
